## Prerequisites

- node >= 10.x && node <= 16.x

## Install with [npm](https://www.npmjs.com/package/image-flow)

```sh
yarn install image-flow
```

## Usage

### base

```javascript
<ImageFlow
  width={400}
  height={690}
  type="Soul"
  url="https://s1.ax1x.com/2023/02/23/pSxaa1U.jpg"
/>
```
